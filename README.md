# 🌾 Crop & Food Product Explorer

## 📌 Project Description

The Crop & Food Product Explorer is a web application that allows users to search and explore food and agricultural products using the Open Food Facts API. The application displays product-related information such as name, category, country of origin, and nutritional details.

## 🎯 Purpose

The purpose of this project is to demonstrate the use of JavaScript, API integration, and UI development while solving a real-world problem of exploring food and agricultural data.

## 🌐 API Used

Open Food Facts API
https://world.openfoodfacts.org/

## ⚙️ Features (Planned)

* 🔍 Search products by name
* 📊 Display product data dynamically
* 🎯 Filter by category or country
* ↕️ Sort alphabetically or by nutrition
* 📱 Responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (ES6)
* Fetch API

## 🚀 How to Run

1. Clone the repository
2. Open index.html in browser
